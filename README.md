processing code for Least Square Super-Resolution Reconstruction

originally from https://github.com/filip-szczepankiewicz/Vis_NIMG_2021/

1. run example_cima_QL_v0.m on pa data first (assume pa only contains 1 dwi volume);
2. run example_cima_QL_v1.m for ap data, all the diffusion volumes will be automatically detected and process
